# 巴基斯坦本地化图片合规检查器

自动检查图片设计是否符合巴基斯坦本地化要求的在线工具。

## 功能特点

- ✅ 4 维度分析（语言、设计、宗教、文化）
- ✅ 智能问题检测（高/中/低严重程度）
- ✅ 详细修改建议
- ✅ 拖拽上传 + URL 输入
- ✅ 无需登录，公开访问
- ✅ 简洁易用的界面

## 技术栈

- **前端**: Next.js 14.2.0 + React 18.2.0
- **AI 分析**: Claude Opus 4.5 (通过 OpenRouter API)
- **部署**: Vercel

## 快速开始

### 本地开发

1. **配置环境变量**

   复制 `.env.example` 为 `.env`：
   \`\`\`bash
   cp .env.example .env
   \`\`\`

   编辑 `.env` 文件，填入你的 OpenRouter API 密钥：
   \`\`\`env
   OPENROUTER_API_KEY=sk-or-v1-你的密钥
   \`\`\`

2. **安装依赖**

   \`\`\`bash
   npm install
   \`\`\`

3. **启动开发服务器**

   \`\`\`bash
   npm run dev
   \`\`\`

4. **访问应用**

   打开浏览器访问 [http://localhost:3000](http://localhost:3000)

### 部署到 Vercel

1. 将代码推送到 GitHub
2. 在 Vercel 中导入项目
3. 配置环境变量：
   - `OPENROUTER_API_KEY`: 你的 OpenRouter API 密钥
4. 部署完成

## 使用方法

1. 选择输入方式（文件上传或 URL 输入）
2. 上传图片或输入图片 URL
3. 点击"开始分析"
4. 查看详细的分析结果和修改建议

## 分析维度

### 1. 语言准确性
- 乌尔都语使用是否正确
- 文字方向（从右到左）
- 字体选择
- 英语拼写和语法

### 2. 设计风格符合度
- 颜色使用（绿色和白色是正面颜色）
- 布局符合当地阅读习惯
- 排版和视觉层次

### 3. 宗教禁忌敏感性
- 伊斯兰教敏感元素
- 猪和酒精相关图像
- 女性形象得体性
- 宗教节日和符号

### 4. 文化敏感性
- 手势含义（左手被认为不洁）
- 符号使用（新月和星）
- 人物形象
- 文化习俗

## 环境变量

| 变量名 | 说明 | 必填 |
|--------|------|------|
| OPENROUTER_API_KEY | OpenRouter API 密钥 | 是 |
| MAX_IMAGE_SIZE_MB | 最大图片大小（MB） | 否 |
| ALLOWED_IMAGE_TYPES | 允许的图片类型 | 否 |

## 常见问题

### Q: 分析需要多长时间？
A: 通常在 5-15 秒内完成，取决于图片大小和复杂度。

### Q: 支持哪些图片格式？
A: 支持 PNG、JPEG、JPG、WebP 格式。

### Q: 图片大小限制是多少？
A: 最大 10MB。

### Q: 分析结果准确吗？
A: 使用 Claude Opus 4.5 进行分析，准确率较高，但建议结合人工复核。

### Q: 我的图片会被存储吗？
A: 不会。图片仅用于分析，分析后立即丢弃，不保存任何记录。

## 注意事项

1. 本工具仅提供参考，建议结合人工复核
2. 分析结果可能因 AI 模型的理解偏差而有所不同
3. 对于重要的商业项目，建议咨询专业的本地化团队
4. 请遵守 OpenRouter API 的使用条款
