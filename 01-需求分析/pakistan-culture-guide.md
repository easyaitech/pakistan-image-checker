# 巴基斯坦文化禁忌参考指南

本指南总结了在进行巴基斯坦本地化设计时需要注意的文化、宗教、语言等方面的禁忌和偏好。

## 语言相关

### 乌尔都语（Urdu）
- **地位**: 巴基斯坦国语
- **文字方向**: 从右到左（RTL）
- **文字系统**: 波斯-阿拉伯文字系统
- **注意事项**:
  - 必须正确设置 RTL 布局
  - 字体需要支持阿拉伯文字系统
  - 拼写和语法要准确
  - 避免机器翻译，建议使用专业翻译

### 英语
- **地位**: 官方语言，广泛使用
- **注意事项**:
  - 拼写和语法要正确
  - 避免使用当地人不熟悉的俚语
  - 专业术语需要准确翻译

## 颜色文化

### 正面颜色
- **绿色**: 伊斯兰教象征，代表繁荣和神圣
  - 巴基斯坦国旗的主要颜色
  - 最受民众欢迎的颜色
- **白色**: 和平、纯洁
  - 国旗的次要颜色
  - 代表宗教少数群体
- **蓝色**: 信任、专业
- **金色/黄色**: 繁荣、财富

### 负面或需谨慎使用的颜色
- **红色**: 在某些语境下有负面含义
  - 不建议大面积使用
  - 可用于强调（如按钮）
- **黑色**: 哀悼、负面（某些语境）
- **黄色**: 某些语境下代表疾病

### 颜色搭配建议
- 绿色 + 白色：最安全的选择
- 绿色 + 金色：高端、专业
- 蓝色 + 白色：商务、信任
- 避免过度使用红色和黑色

## 宗教禁忌（伊斯兰教）

### 严格禁止的内容
1. **先知形象**: 绝对不能描绘先知穆罕默德的形象
2. **猪相关的图像**: 猪肉在伊斯兰教中是禁忌
3. **酒精饮料**: 伊斯兰教禁止饮酒
4. **赌博相关内容**: 伊斯兰教禁止赌博

### 需要注意的内容
1. **女性形象**:
   - 避免过度暴露
   - 建议穿着得体
   - 避免过于性感的姿势

2. **宗教节日**:
   - 斋月（Ramadan）：穆斯林 fasting 的月份
   - 开斋节（Eid al-Fitr）：斋月结束的节日
   - 古尔邦节（Eid al-Adha）：宰牲节
   - 避免在这些节日期间发布不当内容

3. **清真认证**:
   - 食品相关内容需注意是否有清真标志
   - 避免展示非清真食品

4. **清真寺**:
   - 尊重清真寺的图像
   - 避免不当使用

## 文化敏感性

### 手势

### 禁止或需谨慎的手势
1. **左手**: 被认为不洁
   - 避免用左手递东西
   - 避免用左手吃饭
   - 避免用左手打招呼

2. **竖大拇指**: 在某些地区有侮辱性
   - 类似于其他国家的中指
   - 需谨慎使用

3. **OK 手势**: 可能被误解
   - 在某些文化中有负面含义
   - 建议避免

4. **用手指指人**: 粗鲁
   - 使用整只手掌
   - 或用拇指示意

### 推荐的手势
- 右手递物品
- 双手合十（表示尊重）
- 微笑点头

### 符号

### 正面符号
1. **新月和星**: 巴基斯坦国徽元素
   - 伊斯兰教象征
   - 需尊重使用

2. **巴基斯坦国旗**:
   - 绿色和白色
   - 新月和星
   - 需正确使用

3. **茉莉花**: 国花

### 需谨慎使用的符号
- 宗教符号：需了解其含义
- 政治符号：避免争议

### 人物形象

### 服饰
1. **传统服饰**:
   - **Salwar Kameez**: 长衫和宽松裤子
   - **Shalwar**: 宽松裤子
   - **Kameez**: 长衫
   - **Dupatta**: 女性披肩

2. **头巾**:
   - 男性头巾（Pagri）
   - 尊重头巾文化

### 性别互动
- 男女之间保持适当距离
- 避免过于亲密的肢体接触
- 女性形象需得体

## 设计偏好

### 布局
- **对称**: 偏好对称和秩序感
- **从右到左**: 乌尔都语内容的布局方向
- **视觉层次**: 清晰的信息层级

### 排版
- **字体**: 需支持阿拉伯文字系统
- **字号**: 适中的字号，便于阅读
- **行距**: 充足的行距，提高可读性

### 图案
- **几何图案**: 传统纹样和几何图案
- **花卉图案**: 茉莉花等传统图案
- **传统艺术**: Islamic art 风格

### 图片风格
- **明亮色彩**: 偏好饱和、明亮的色彩
- **真实人物**: 真实的人物形象，避免过度美化
- **家庭场景**: 家庭和睦的场景受欢迎

## 特定场景的注意事项

### 食品相关
- 必须显示清真标志
- 避免展示猪肉、酒精
- 避免 showing people eating with left hand

### 教育相关
- 尊重学习场景
- 避免不当的性别刻板印象
- 显示多元化的学习环境

### 医疗相关
- 尊重文化差异
- 避免不当的性别展示
- 注意医患互动的得体性

### 金融相关
- 避免展示赌博、高利贷等
- 符合伊斯兰教法（Islamic Banking）
- 避免显示 interest（Riba）

## 节日和重要日期

### 宗教节日
1. **斋月（Ramadan）**: 伊斯兰历第九月
2. **开斋节（Eid al-Fitr）**: 斋月结束后
3. **古尔邦节（Eid al-Adhu）**: 伊斯兰历第十二月
4. **伊斯兰历新年**: Muharram

### 国庆日
- **独立日**: 8月14日（1947年）
- **巴基斯坦日**: 3月23日

### 营销建议
- 在重要节日期间推出特别活动
- 尊重宗教节日的神圣性
- 避免在斋月白天展示美食图片

## 测试清单

在设计审查时，使用以下清单进行检查：

### 语言检查
- [ ] 乌尔都语文字方向是否正确（RTL）
- [ ] 字体是否支持阿拉伯文字系统
- [ ] 英语拼写和语法是否正确
- [ ] 文字与背景对比度是否足够

### 设计检查
- [ ] 颜色是否符合当地文化（绿色、白色为正面）
- [ ] 布局是否合理
- [ ] 排版是否清晰
- [ ] 图片风格是否贴近当地审美

### 宗教检查
- [ ] 是否包含伊斯兰教敏感元素
- [ ] 是否有违反伊斯兰教义的图像
- [ ] 女性形象是否得体
- [ ] 是否尊重宗教节日和符号

### 文化检查
- [ ] 手势是否得体（避免左手、竖大拇指等）
- [ ] 符号使用是否正确
- [ ] 人物形象是否符合当地习俗
- [ ] 是否有文化误解风险
